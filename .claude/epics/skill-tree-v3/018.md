# Task: Hex grid layout for 57 skills

## Description

Completely redesign `SKILL_POSITIONS` in `hex-grid-layout.ts` to place all 57 skills on the hex grid. The critical constraint: every prerequisite pair MUST share a hex edge (be adjacent in axial coordinates).

Cross-branch T2 skills must sit at the boundary between their two parent branch sectors. T3 skills form ring 4 in 6 zones of 4 skills each.

## Acceptance Criteria

- [ ] `SKILL_POSITIONS` contains entries for all 57 skill IDs
- [ ] Every prerequisite pair (for all skills) passes `areAdjacent()` check
- [ ] T0 base skills remain near center
- [ ] T1 skills radiate outward from their T0 parent
- [ ] Boost passives (sharpen, swift-feet, iron-skin) are adjacent to their T0 parent
- [ ] Within-branch T2 skills are adjacent to BOTH their T1 prerequisites
- [ ] Cross-branch T2 skills sit between the two branch sectors of their parents
- [ ] T3 skills on outer ring, each adjacent to both its T2 and T1 prerequisites
- [ ] No two skills occupy the same coordinate
- [ ] Grid is roughly symmetric across the 3 branch sectors (120deg each)
- [ ] Adjacency verification script/test confirms all constraints pass

## Technical Details

### Files affected
- `src/lib/hex-grid-layout.ts` — Rewrite `SKILL_POSITIONS` record

### Implementation approach

1. Start with T0 at center triangle (keep existing: melee-attack at (0,-1), move at (1,0), defend at (-1,1))
2. Place T1 skills in ring 1-2, maintaining branch sectors:
   - Attack sector: north/upper-right
   - Movement sector: east/lower-right
   - Defend sector: west/lower-left
3. Place boost passives adjacent to their T0 base, off to the side (not blocking main progression)
4. Place within-branch T2 skills at ring 2-3, each adjacent to both T1 parents
5. Place cross-branch T2 skills at sector boundaries:
   - AD hybrids: between Attack and Defend sectors
   - AM hybrids: between Attack and Movement sectors
   - DM hybrids: between Defend and Movement sectors
6. Place T3 skills on ring 4, grouped into 6 zones of 4

### Layout strategy

The hex grid has 6 natural "wedge" directions. With 3 branches at 120deg spacing:
- Attack sector occupies ~120deg (roughly the "north" wedge)
- Movement sector occupies ~120deg (roughly the "east/southeast" wedge)
- Defend sector occupies ~120deg (roughly the "west/southwest" wedge)
- Between each pair of sectors: a boundary zone for cross-branch hybrids

### Adjacency verification

Add a verification function or test at the bottom of the file:
```typescript
export function verifyAdjacency(): string[] {
  const errors: string[] = [];
  for (const skill of SKILLS) {
    const pos = SKILL_POSITIONS[skill.id];
    if (!pos) { errors.push(`Missing position for ${skill.id}`); continue; }
    for (const prereqId of skill.prerequisites) {
      const prereqPos = SKILL_POSITIONS[prereqId];
      if (!prereqPos) { errors.push(`Missing position for prereq ${prereqId}`); continue; }
      if (!areAdjacent(pos, prereqPos)) {
        errors.push(`${skill.id} not adjacent to prereq ${prereqId}`);
      }
    }
  }
  return errors;
}
```
