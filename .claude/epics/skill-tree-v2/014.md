# Task: Advantage Legend

## Description

Add a small SVG legend inside the graph explaining what gold and purple highlights mean. The legend should be non-intrusive, readable, and pan/zoom with the graph (rendered inside the SVG, not as HTML overlay).

## Acceptance Criteria

- [ ] Legend renders inside the SVG viewBox (pans and zooms with the graph)
- [ ] Shows a small gold hex icon + "Beats" (or "Advantage")
- [ ] Shows a small purple hex icon + "Vulnerable" (or "Weak to")
- [ ] Positioned near the outer edge of the graph (lower-left corner of the hex ring area)
- [ ] Non-intrusive — doesn't overlap with skill nodes
- [ ] Readable at default zoom level
- [ ] Uses semi-transparent background for contrast
- [ ] `bun run build` passes with 0 errors

## Technical Details

### Files Affected

1. **`src/components/skill-tree/UnifiedSkillGraph.tsx`** — only file changed

### Implementation

Add an SVG `<g>` group after the skill nodes but before the confirmed outlines. Position it using the outermost hex radius from `hexRadii`:

```tsx
{/* Advantage Legend */}
<g transform={`translate(${-legendX}, ${legendY})`} style={{ pointerEvents: "none" }}>
  {/* Background */}
  <rect x={-8} y={-8} width={120} height={50} rx={6}
    fill="#030712" fillOpacity={0.8} stroke="#ffffff" strokeWidth={0.5} strokeOpacity={0.2} />

  {/* Gold = Beats */}
  <polygon points={hexPoints(-0, 8, 8)} fill={ADVANTAGE_COLOR} fillOpacity={0.6}
    stroke={ADVANTAGE_COLOR} strokeWidth={1} />
  <text x={16} y={12} fill="#fbbf24" fontSize={8} fontWeight="600">Beats</text>

  {/* Purple = Vulnerable */}
  <polygon points={hexPoints(0, 30, 8)} fill={DISADVANTAGE_COLOR} fillOpacity={0.6}
    stroke={DISADVANTAGE_COLOR} strokeWidth={1} />
  <text x={16} y={34} fill="#a855f7" fontSize={8} fontWeight="600">Vulnerable</text>
</g>
```

Position calculation:
```typescript
const legendX = hexRadii.length > 0 ? hexRadii[hexRadii.length - 1] + 20 : 200;
const legendY = legendX * 0.6; // lower-left area
```

### Design Notes

- Keep it small (8px hex icons, 8pt text)
- Use the same `ADVANTAGE_COLOR` and `DISADVANTAGE_COLOR` constants from Task 002
- Semi-transparent dark background so it's readable against both dark background and any hex ring outlines
- No interaction (pointerEvents: none)
