# Task: Project Scaffolding

## Description

Scaffold the Next.js 15 project at `~/Projects/tree` with TypeScript, Tailwind CSS v4, and App Router. Install dependencies (zustand, zod), create the full directory skeleton, initialize git, and write CLAUDE.md with project conventions. This is the foundation — every other task depends on it.

## Acceptance Criteria

- [ ] Next.js 15 app created with `bun create next-app` using TypeScript, Tailwind, App Router, ESLint
- [ ] Dependencies installed: `zustand@^5`, `zod@^4`
- [ ] `bun dev` serves the app at localhost:3000 without errors
- [ ] Directory skeleton created:
  - `src/data/`, `src/types/`, `src/lib/`, `src/stores/`
  - `src/components/skill-tree/`, `src/components/skill-card/`, `src/components/build/`, `src/components/ui/`
  - `src/hooks/`
  - `src/app/print/`
- [ ] `tsconfig.json` has `@/` path alias pointing to `src/`
- [ ] `next.config.ts` configured with `output: 'export'` for static deployment
- [ ] `CLAUDE.md` created with project conventions (stack, structure, naming, colors)
- [ ] Git repository initialized with initial commit
