# Task: Print View, Direction Line + Polish

## Description

Final task: add the `/print` route with a printable card sheet, the directional cursor line with branch highlighting, SVG pan/zoom, responsive layout adjustments, and visual polish (glow effects, edge animations, keyboard navigation). This brings the app to v1 feature-complete.

## Acceptance Criteria

### Print View
- [ ] `/print` route renders a `CardSheet` component
- [ ] CardSheet shows only unlocked skills from the active build
- [ ] Cards rendered at poker-card proportions (63mm x 88mm) on white background
- [ ] CSS Grid layout: 3 columns, cards organized by tier then branch
- [ ] `@media print` styles: white background, no sidebar, no navigation, proper margins
- [ ] `break-inside: avoid` prevents cards from splitting across pages
- [ ] Cmd+P in Chrome produces clean, cuttable output
- [ ] "Back to Tree" link navigates to `/`

### Direction Line
- [ ] `use-mouse-angle` hook tracks cursor angle relative to SVG center
- [ ] `DirectionLine` renders an SVG line from (0,0) to cursor position (clamped to outer ring radius)
- [ ] Line is thin, semi-transparent white
- [ ] Nearest branch slice is highlighted (full opacity), other slices dimmed
- [ ] Nearest individual skill node gets a subtle glow effect
- [ ] Highlighting updates smoothly at 60fps

### Pan/Zoom
- [ ] Mouse wheel zooms the SVG (scales viewBox)
- [ ] Click-drag pans the SVG (translates viewBox)
- [ ] Zoom reset button (or double-click to reset)
- [ ] Zoom range clamped (0.5x to 3x)

### Responsive
- [ ] On screens narrower than 768px, sidebar collapses to a bottom drawer
- [ ] Tree remains usable on tablet viewports
- [ ] Touch: pinch-to-zoom, swipe-to-pan

### Polish
- [ ] SVG `<defs>` glow filter applied to unlocked nodes
- [ ] Edge animation: `stroke-dashoffset` animation for "flowing" prerequisite lines on active edges
- [ ] Keyboard navigation: Tab through nodes, arrow keys traverse the graph, Enter toggles unlock
- [ ] SVG nodes have `aria-label` with skill name for accessibility
- [ ] `TreeLegend` component in bottom corner showing branch color key
- [ ] Export Build: "Copy to clipboard" button serializes build as JSON
- [ ] Import Build: "Paste build" loads a JSON build into the current slot

### Direction Line Algorithm
```typescript
// use-mouse-angle hook:
// 1. Get SVG element ref
// 2. On mousemove, compute cursor position relative to SVG center
// 3. Convert to angle: atan2(dx, -dy) (adjusted for SVG coords)
// 4. Find nearest branch by comparing angle to BRANCH_ANGLES
// 5. Find nearest skill node by angular + radial proximity
```

### Pan/Zoom
- Store `viewBox` as state: `{ x, y, width, height }`
- Wheel: adjust width/height proportionally (zoom center at cursor)
- Drag: adjust x/y based on movement delta scaled by current zoom
